# JS assessment

## Setup steps:

### 1. `npm install`

### 2. Create `.env ` file

### 3. Fill the `.env` file with variables from .env.example

### 4. Run `npm start`
### 5. Open `http://localhost:3000` if it doesn't open automatically
## Available Scripts

### `npm start`

### `npm test`

### `npm run build`

### `npm run format`
### `npm run eject`
